1. Project Overview
This project is a web-based Islamic Quiz application developed using HTML, CSS, and JavaScript.
The purpose of the project is to enhance Islamic knowledge in an interactive and user-friendly way.
Users can select quiz categories, answer multiple-choice questions, and receive instant feedback
with explanations.

3. Technologies Used
1 HTML: Used to create the structure of the quiz application.
2 CSS: Used for styling, layout, background image, and text visibility.
3 JavaScript: Used to implement quiz logic, category handling, scoring, and dynamic content
updates.

5. Project Features
1 Multiple quiz categories (Pillars of Islam, Prophets, Quran, History of Islam).
2 Dynamic question loading based on selected category.
3 Multiple-choice questions with instant answer validation.
4 Score calculation and result summary.
5 Explanation displayed after each answer.
6 User-friendly interface with background image.

7. Category Implementation
The quiz categories are implemented using a JavaScript object named questionBank. Each
category contains an array of question objects. Each question object includes the question text, four
options, the correct answer index, and a brief explanation.

9. Quiz Logic
1 User selects a category using category buttons.
2 The selected category determines which questions are loaded.
3 Questions are displayed one by one.
4 User selection is validated immediately.
5 Correct and incorrect answers are highlighted.
6 Score is updated dynamically.
7 Final result is displayed at the end of the quiz.

11. User Interface Design
The interface uses a background image for visual appeal. Text color is adjusted to white for better
readability. Content visibility is controlled using JavaScript to show and hide different sections
dynamically.

13. Conclusion
This Islamic Quiz project demonstrates the practical use of HTML, CSS, and JavaScript to build an
interactive educational application. The project is easy to extend by adding more categories or
questions in the future
